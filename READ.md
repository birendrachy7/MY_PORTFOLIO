#  Getting Started

### Clone the repository

```bash
git clone https://github.com/birendrachy7/Portfolio.git
```

### Navigate into the project

```bash
cd Portfolio
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
